# SCREEN Demo

This repository contains a real-time 4x super-resolution demo running on the AMD Kria KV260.

[View the demo video](./SCREEN_demo.mp4)

## Demo Overview

The KV260 processes a 180 x 320 input video and generates a 720 x 1280 output video in real time.

- **Left:** The original 180 x 320 input, enlarged using simple spatial upscaling only for an easier visual comparison. It is not the super-resolved output.
- **Right:** The 720 x 1280 super-resolved output generated by the KV260.
- **Frame rate:** The recorded demonstration is limited to 60 FPS due to the frame-rate limitation of the camera used for recording. The 60 FPS shown in this video does not represent the maximum throughput of the accelerator.
